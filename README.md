# 📌 Ecommerce Website "Kixa"
Full stack e-commerce website. I started this project after learning HTML, CSS and JavaScript basics. My goal was to learn real-world development practices. Due throwing myself into a challenging but achievable project with limited web-development knowledge, I had to often rewrite and rethink major parts of the application. I'm not finished yet, but I've already made huge progress compared the first version of the website.

## 🚀 Demo  
[Live Demo](https://22-nicolas.github.io/kixa)  

notes: 
- The application is hosted on free-tier services, so the backend may go to sleep after inactivity and take some time to respond to the first request. The database may also shut down after extended periods of inactivity.
If you encounter any issues while using the application, please feel free to contact me.
- There are some styling inconsistencies with firefox.

## ✨ Features

-  **React Frontend** – Modern UI built with React for a dynamic and fast user experience  
-  **Bootstrap Styling** – Responsive and consistent layout using Bootstrap components  
-  **Responsive Design** – Optimized for different screen sizes and devices  
-  **Secure Authentication** – User registration and login system with secure handling of credentials  
-  **Product Search & Filters** – Easily search and filter products to find what you need quickly  
-  **Database Integration** – Backend connected to a database for storing user and product data
-  **Phone Number API Integration** – Registration page uses the aaapis.com API to automatically detect the country prefix and display the correct flag  
-  **Shopping Cart** – Add, remove, and manage products before checkout  
-  **Deployed Application** – Fully deployed and accessible online
